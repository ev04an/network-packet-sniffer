# Network Packet Sniffer
A Python-based packet sniffer that captures and analyzes network packets.

## 🚀 Features:
✅ Captures TCP, UDP, and IP packets  
✅ Saves packets to a CSV file  
✅ Filters packets by protocol  
✅ GUI to display packets in real-time
